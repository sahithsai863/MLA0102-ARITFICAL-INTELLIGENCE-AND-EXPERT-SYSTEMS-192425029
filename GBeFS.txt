START

Create a priority queue.

Insert the Robot Start position into the priority queue using its heuristic value.

Mark all locations as unvisited.

WHILE the priority queue is not empty DO

    Remove the location with the smallest heuristic value.

    Display the current location.

    IF current location is the Target Rack THEN
        STOP.

    Mark the location as visited.

    FOR each neighboring location DO
        IF the neighbor has not been visited THEN
            Insert it into the priority queue based on its heuristic value.
        END IF
    END FOR

END WHILE

STOP
